## Slide 3: Pertemuan Pertama dengan Linux
### *"Jatuh Cinta Pada Pandangan Pertama"*

**Tahun:** [2010/2015/2020]
**Situasi:**
- Komputer lama yang perlahan dengan Windows

## Slide 4: Kejutan Budaya Pertama
### *Detik-detik Menyedari Realiti*

**Yang Mengejutkan:**
- Tak payah beli lesen Windows RM500+
- Repository perisian - pasang dengan satu klik
- Terminal yang berkuasa (menakutkan pada mulanya!)
- Komuniti yang membantu dan mesra
- Kemaskini sistem tak paksa restart berkali-kali

**Yang Mencabar:**
- Masalah keserasian permainan
- Pembelajaran command line
- Driver perkakasan kadang-kadang bermasalah

## Slide 5: Evolusi Penggunaan Linux
### *Dari Pemula kepada Peminat*

**Fasa 1: Pemula Ubuntu** (2-6 bulan)
- Kekal dengan tetapan lalai
- Takut sentuh terminal
- Dual boot dengan Windows "untuk berjaga-jaga"

**Fasa 2: Melompat Distro** (6 bulan - 2 tahun)
- Mencuba Fedora, openSUSE, Mint
- Belajar penyesuaian
- Mula menghargai command line

**Fasa 3: Asli Linux** (2+ tahun)
- Windows jadi "OS sandaran"
- Selesa dengan terminal
- Menyumbang kepada projek open source

## Slide 8: Fedora - Pengalaman Terdepan
### *Edisi Komuniti Red Hat*

**Kenapa Fedora?**
- Teknologi terkini dahulu

- Fokus keselamatan

- Varian KDE, XFCE, LXQt

## Slide 16: Mitos vs Realiti
### *Menyanggah Mitos Linux*

**MITOS:** "Linux susah untuk pengguna bukan teknikal"
**REALITI:** Ubuntu sama mudah dengan Windows untuk tugas asas

**MITOS:** "Tak ada perisian untuk Linux"
**REALITI:** 99% perisian harian ada alternatif yang bagus

**MITOS:** "Linux tak stabil untuk pengeluaran"
**REALITI:** Majoriti pelayan web guna Linux

**MITOS:** "Tak boleh main game di Linux"
**REALITI:** Steam Proton, Wine, permainan asli semakin banyak

## Slide 20: Tips untuk Pemula
### *Panduan Permulaan*

**Langkah 1:** Mulakan dengan Ubuntu atau Linux Mint
**Langkah 2:** Dual boot dulu, jangan terus padam Windows
**Langkah 3:** Sertai komuniti tempatan (dalam talian/luar talian)
**Langkah 4:** Amalkan command line sedikit-sedikit
**Langkah 5:** Terokai alternatif perisian

**Peraturan Emas:**
- Sandarkan data sebelum eksperimen
- Google adalah kawan terbaik anda
- Jangan takut bertanya
- Nikmati perjalanan, bukan destinasi sahaja

